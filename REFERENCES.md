# Observational and Theoretical References Used in CoDE-4

The references below represent the primary observational and theoretical sources used for consistency comparison throughout the project.

# Reference Usage

The CoDE-4 framework uses the above references primarily for:
- observational comparison,
- approximate consistency evaluation,
- baseline ΛCDM benchmarking,
- and phenomenological interpretation.

The project does not currently perform:
- full survey-likelihood fitting,
- Bayesian parameter estimation,
- or direct MCMC statistical inference against these datasets.

All comparisons should therefore be interpreted as exploratory computational consistency studies rather than precision cosmological parameter fits.
